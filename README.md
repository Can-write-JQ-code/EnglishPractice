# 英语练习应用程序

这是一个基于Python的英语练习应用程序，提供以下核心功能：
1. 认识单词：随机展示英语单词及其中文释义，支持自动切换和语音朗读
2. 单词填空：提供部分字母缺失的单词，用户需填写缺失字母
3. 中文盲打：显示中文释义，用户需输入对应的英语单词

## 安装与运行

1. 安装依赖：
```
pip install -r requirements.txt
```

2. 启动应用：
```
python app.py
```

3. 在浏览器中访问：
```
http://localhost:5000
```

## 技术栈
- 后端：Flask (Python)
- 前端：HTML, CSS, JavaScript
- 数据库：MongoDB
